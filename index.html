So you think you know...

This is a simple quiz game to test your knowledge of the Force and the Star Wars Universe.
You will find some questions that test your knowledge of the Legends version of the Star Wars Universe and the Canon version.

Features

Highscores

Compete with the best in the galaxy to see where you stand against the rest!

Simple and Fun

Fun and engaging quiz page, simple and easy quiz page for a great User Experience.

Play and learn

Learn more about the Star Wars Universe from responsive answer selection.

Instructions

Simply click the Play Now button below to start the quiz. Select the answer you think is correct, correct answers will turn green, incorrect answers will turn red. On the last page, type a name to save your high score.
